  sessionID

Session ID of the current recognition transaction.

@property (nonatomic, readonly) NSString *sessionID

Discussion

The session ID is only valid after the session is established with the server. If nil, the session is not (yet) established. Also note that multiple transactions can take place in the same session (hence same ID).
